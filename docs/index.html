데스크포털 소스 허브

이 저장소는 데스크포털의 GitHub 원본입니다. 실제 서비스는 Google Apps Script 웹앱으로 운영하고, 이 페이지는 프로젝트 소개, 소스 구조, 운영 링크 안내를 위한 정적 진입 페이지로 사용합니다.

GitHub 저장소 열기 Apps Script 소스 보기 운영 웹앱 URL 연결 예정

현재 구성

데스크포털은 수강료 정산, 강사 시수 정산, 운영 관리 성격의 도구를 담는 내부 포털입니다. GitHub는 코드 원본과 변경 이력을 관리하고, 실제 사용자 실행은 GAS 웹앱에서 이어집니다.

Module
재원생 수강료 정산
수납 입력, 최근 수납, 월별 매출, 안내 대시보드, 학생 월별 수강료 이력 관리
Module
강사 시수 정산
출결, 인정 시수, 정산 방식, 월별 요약과 상세 테이블을 함께 다루는 정산 포털
Workflow
GitHub → GAS
코드는 GitHub에서 관리하고, 필요 시 canonical 파일과 Apps Script 편집기로 반영합니다

주요 경로

현재 Pages는 정적 안내 페이지이고, 실제 앱 코드는 아래 경로를 기준으로 관리합니다.

Apps Script Source
apps-script/payroll/Code.gs apps-script/payroll/Index.html
GitHub Pages
docs/index.html
Sync Scripts
scripts/sync_from_canonical.sh scripts/sync_to_canonical.sh
운영 웹앱 URL이 준비되면 이 파일 상단의 DESK_PORTAL_WEBAPP_URL 값만 채우면 됩니다. 그러면 상단 버튼이 실제 GAS 웹앱 링크로 바뀝니다.
